# Mythology & Worldbuilding Linting and Best Practices Enforcer VS Code Extension

> Custom rules, auto-fix, inline warnings, and team config sharing for Mythology & Worldbuilding code. Catches Mythology & Worldbuilding-specific bugs generic linters miss.

## Features

- Core functionality

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.7dc3c850`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Mythology & Worldbuilding Linting and Best Practices Enforcer VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `7dc3c850.enabled` | `true` | Enable/disable the extension |
| `7dc3c850.autoRun` | `false` | Auto-run on file open |
| `7dc3c850.logLevel` | `"info"` | Log verbosity |
